# 技术选型

本项目旨在构建一个现代化、高性能且易于维护的 SaaS 数字孪生平台。为此，我们选择了以下技术栈：

### 核心框架

- **Vue.js 3**: 作为项目的基础框架。我们利用其组合式 API (Composition API) 来组织代码，提高逻辑复用性和可读性。其响应式系统和组件化思想非常适合构建复杂的用户界面。

- **Vite**: 用于项目构建和开发环境。Vite 提供了闪电般的冷启动速度和模块热更新 (HMR) 能力，极大地提升了开发效率。

### 状态管理

- **Pinia**: Vue 的官方钦定状态管理库。它拥有极简的 API 设计，与 Vue 3 的组合式 API 完美契合，并提供完善的 TypeScript 支持，使状态管理变得类型安全和直观。

### UI 组件库

- **Element Plus**: 一套基于 Vue 3 的企业级 UI 组件库。它提供了丰富、高质量的组件，可以帮助我们快速构建出美观、一致的界面，而无需从零开始造轮子。

### 路由管理

- **Vue Router**: Vue 的官方路由管理器。它与 Vue 核心深度集成，可以轻松地实现页面间的导航和参数传递。

### 画布拖拽与交互

- **VueDraggable**: 基于强大的 `Sortable.js`，我们将用它来实现从物料面板拖拽组件到画布的功能。

- **interact.js**: 一个功能强大且灵活的、与框架无关的交互库。我们将用它来实现画布上组件的拖拽、缩放、旋转等核心交互。其强大的修饰符 (modifiers) API 将帮助我们实现对齐、吸附和辅助线等高级功能。

### 包管理器

- **pnpm**: 我们选用 pnpm 作为包管理器。它通过创新的符号链接方式来管理 `node_modules`，能够显著节省磁盘空间，并提升安装速度。
